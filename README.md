# Laravel-API-Offers
Simple Laravel API with CRUD operations on RCA quotes

# API Endpoints
- `GET` `/api/quotes`: retrieve all quotes
- `GET` `/api/quotes/{id}` retrieve a quote by its ID
- `POST` `/api/quotes` add a new quote
- `PUT` `/api/quotes/{id}` fully update a quote by its ID
- `PATCH` `/api/quotes/{id}` partially update a quote by its ID

# Database
- `sqlite`
